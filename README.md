🚀 DigiTools — Premium Digital Marketplace

## 📌 About the Project
**DigiTools** is a modern, responsive web application designed as a digital marketplace for premium tools. It allows users to explore curated digital products, manage their cart efficiently, and enjoy a smooth, visually appealing user experience.
The platform focuses on simplicity, performance, and productivity — making it ideal for creators, developers, and professionals.

---

## ⚙️ Technologies Used
### 🧩 Frontend
* ⚛️ React.js
* ⚡ Vite
### 🎨 Styling & UI
* 🌬️ Tailwind CSS
* 🌼 DaisyUI
### 🔧 Tools & Libraries
* 🔔 React Toastify (Notifications)
* 🎯 Lucide React (Icons)
### 📦 Data Handling
* JSON-based dynamic product system
### 🧠 Language
* JavaScript (ES6+)

## ✨ Key Features
### 🛒 Smart Cart System

* Duplicate item prevention

### 📦 Dynamic Product Rendering
* Products loaded from JSON data
* Scalable and easily maintainable structure

### 🎨 Modern UI/UX Design
